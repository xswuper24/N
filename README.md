## Build

### Dependências
- [Node.js](https://nodejs.org/en/download/)
- [grunt-cli](https://github.com/gruntjs/grunt-cli)

```bash
git clone https://github.com/mafrazz/tw2autofarm.git
cd tw2autofarm
npm install
grunt
```
